# PDF 朗读器

一个简单易用的PDF文件朗读工具，上传PDF文件即可自动朗读内容。

## 功能特点

- 支持上传PDF文件（点击或拖拽）
- 自动提取PDF中的文本内容
- 使用浏览器内置语音引擎朗读
- 可选择不同的语音（支持中文、英文等）
- 可调节语速（0.5x - 2x）
- 可调节音调（0.5 - 2）
- 播放控制：播放/暂停/停止
- 实时显示朗读进度
- 文本内容预览

## 运行方法

### 方法1：使用Python启动本地服务器（推荐）

```bash
# 在项目目录下执行
python -m http.server 8000
```

然后在浏览器中打开：http://localhost:8000

### 方法2：直接打开HTML文件

双击 `index.html` 文件，在浏览器中打开即可。

> 注意：某些浏览器可能会限制本地文件的某些功能，建议使用方法1。

### 方法3：使用其他HTTP服务器

```bash
# 使用 Node.js 的 http-server
npx http-server -p 8000

# 使用 PHP
php -S localhost:8000
```

## 使用说明

1. **上传PDF**
   - 点击上传区域选择PDF文件
   - 或直接拖拽PDF文件到上传区域

2. **等待解析**
   - 应用会自动提取PDF中的文本内容
   - 解析完成后会显示文件信息和文本预览

3. **设置朗读参数**
   - 选择语音：从下拉菜单中选择喜欢的语音
   - 调节语速：使用滑块调整朗读速度
   - 调节音调：使用滑块调整声音音调

4. **开始朗读**
   - 点击"播放"按钮开始朗读
   - 点击"暂停"按钮暂停朗读
   - 点击"停止"按钮停止朗读并重置进度

## 技术栈

- **PDF.js** - Mozilla PDF解析库
- **Web Speech API** - 浏览器原生文本转语音API
- **纯前端实现** - 无需后端服务器

## 浏览器兼容性

需要支持以下特性的现代浏览器：
- Web Speech API (SpeechSynthesis)
- ES6+ 语法
- File API

推荐使用：
- Chrome 33+
- Edge 14+
- Safari 7+
- Firefox 49+
